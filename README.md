
# Python Assignment

## 1. Grade Checker
This program takes a score as input and prints the grade using if-else statements.

## 2. Student Grades
This program uses a dictionary to store student names and grades. It allows adding a new student, updating an existing student’s grade, and printing all student grades.

## 3. Write to a File
This program creates a text file and writes content into it using `open()` and `write()`.

## 4. Read from a File
This program opens a file in read mode and reads the content using `read()` and displays it.
